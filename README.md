# mochsner.github.io
This website is based on Jekyll.

## Jekyll / Gems (Ruby)
* Test:
- rake test
* Build:
- bundle exec jekyll serve

* Update packages:
- bundle update github-pages
- bundle update (all gems will update to latest)
